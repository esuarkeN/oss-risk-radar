# MVP Roadmap

## Milestone 1

- monorepo scaffold
- explicit contracts
- mocked end-to-end analysis flow
- dashboard overview and dependency detail pages

## Milestone 2

- real manifest parsing for `package-lock.json`, `requirements.txt`, and `go.mod`
- Postgres-backed persistence in the Go API
- provider adapters for deps.dev, GitHub, and Scorecard

## Milestone 3

- snapshot persistence for repository and scorecard evidence
- richer dependency graphs and transitive resolution
- analyst filtering and saved comparison views

## Milestone 4

- dataset assembly pipeline
- time-aware model training and calibration
- baseline inference registry and model metadata
